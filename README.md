# ⌚ eWatch Landing Page (Figma to Code)

A speed-driven conversion of a Figma landing page into a responsive HTML/CSS prototype.  
Built using **pure HTML5 and CSS3** with a focus on **visual accuracy, responsiveness, and clean structure**.

---

## 🚀 Overview

This project was completed as a **frontend implementation challenge**, translating a static Figma design into a polished landing page using only:

- HTML5
- CSS3 (Flexbox + Absolute Positioning)

No frameworks or libraries were used — only core frontend fundamentals.

---

## 🎯 Key Features

- ⚡ Clean **two-column hero layout** with content and image composition
- 📱 Fully responsive behavior for tablets and mobile screens
- 🎨 Modern UI with consistent spacing and readable typography
- 🧩 Structured HTML hierarchy for better maintainability
- 🖼️ Layered image composition with floating visuals and badges
- ✨ Smooth CTA hover interactions and button states

---

## 🧱 Structure

- **main-wrapper**
  - **content-wrapper**
    - header-container
      - logo-container
      - text-content-container
    - list-wrapper
  - **images-main-wrapper**
    - image-text-content-wrapper
    - main-image-container
    - floating image containers
    - decorative star elements

---

## 📱 Responsive Design

Responsive breakpoints were added to ensure a strong layout across screen sizes:

- `1024px` — refined spacing, scaled imagery, and adjusted overlays
- `768px` — stacked hero layout, static logo position, and fluid image section
- `540px` — mobile-friendly typography, full-width CTAs, and simplified card layout

This improves usability on tablet and mobile devices while preserving the original desktop design.

---

## 🖼️ Image Composition

The visuals were created using:

- precise absolute positioning
- z-index layering for depth
- floating background shapes and overlays
- decorative spark and star imagery

This mirrors the visual hierarchy from the Figma reference.

---

## 🎨 Styling Approach

- Flexbox for layout alignment
- Absolute positioning for controlled overlays
- Responsive breakpoints for smaller screens
- Consistent spacing system
- Subtle shadows and hover effects for interaction
